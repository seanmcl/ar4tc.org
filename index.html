About the Workshop

The Workshop on Automated Reasoning for Tensor Compilers (ARTC) brings together researchers and practitioners working at the intersection of formal methods, program verification, automated reasoning, and tensor compiler technology. As machine learning systems become increasingly critical in various domains, there is a growing need for tensor compilers that are correct, robust, and efficient. This workshop focuses on applying techniques from automated reasoning and logic to address these challenges.

The workshop will feature invited talks, contributed papers, and panel discussions on topics including but not limited to:

  • Formal verification of tensor compiler optimizations
  • Automated reasoning techniques for tensor program analysis
  • Correctness guarantees for tensor computation graphs
  • Formal specification of tensor operations
  • SMT solving for tensor compiler optimization
  • Automated proof generation for tensor transformations
  • Type systems and static analysis for tensor programs
  • Verified code generation for accelerators

Call for Papers

We invite submissions on all aspects of automated reasoning for tensor compilers. Topics of interest include, but are not limited to:

  • Program verification for tensor compilers
  • Automated reasoning for compiler correctness
  • Formal methods for tensor program optimization
  • Symbolic execution of tensor programs
  • Verification of compiler transformations
  • Automated theorem proving for tensor algebra
  • SMT solving for tensor computations
  • Formal guarantees for tensor code generation
  • Type systems for tensor programs
  • Certified compilation for ML frameworks

Submission format: Papers shall be submitted in form of an extended abstract of up to 6 pages in LNCS format (not including references) via email to teme@amazon.com. Submissions can overlap with previously published work and will be judged based on their relevance to the topic of the workshop.
